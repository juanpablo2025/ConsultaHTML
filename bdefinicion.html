Dominio

Un dominio en Internet es el nombre exclusivo y único que se le da a un sitio web para que cualquier internauta pueda visitarlo e identificarlo. Si sabes elegir el adecuado, tu estrategia de marketing digital marchará por excelente camino.

Subdominio

Un subdominio es una forma de tener un sitio (web) relacionado, como anexo, a una web principal. Los subdominios son del tipo: http://subdominio.dominio.com, éstos apuntan realmente a una carpeta del propio alojamiento que has contratado, pero mostrando su contenido desde el subdominio.

Extension de dominio

Una extensión de dominio o dominio de nivel superior (TLD), es la terminación de caracteres alfabéticos que se encuentran después del punto y al final de la dirección de un sitio web.

Hosting

Un hosting web es un servicio que permite que tu página web esté disponible en internet. Existen diferentes tipos de hosting en colombia, como por ejemplo el hosting compartido. Si bien muchos proveedores ofrecen paquetes similares, mientras más popular sea un sitio web, más recursos necesitará para funcionar efectivamente.

Tipos de hosting

  • Hosting gratuito
  • Hosting compartido
  • VPS(Servidores Privados Virtuales)
  • Cloud Hosting o Alojamiento Web en “la nube”
